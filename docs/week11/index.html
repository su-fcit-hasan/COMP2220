Week 11 - mysql with php

Excercise 1 - week11

  • MySQL is an open-source powerful database management system that can handle large amounts of data.
  • This makes MySQL a scalable solution for storing and managing data.
  • MySQL is suitable for data manipulation and retrieval through web applications.

Excercise 2 - week11

  • Data can be manipulated in a database using SQL queries.
  • SQL queries can be used to insert, update, delete, and retrieve data from a database.
  • SQL queries can be used to create, update, and delete tables and databases.

Excercise 3 - week11

  • MySQL can be connected with PHP using the mysqli extension by creating an object of the mysqli class using the new keyword and passing the host, username, password, database name, and port number as arguments to the constructor.
