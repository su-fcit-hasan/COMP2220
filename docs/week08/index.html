1- What is JavaScript, and why is it important in web development?

Javascript provides interactivity webpages and compatibility with all browsers.

2 - How does JavaScript differ from HTML and CSS?

JavaScript is primarily responsible for adding interactivity and dynamic behavior to a webpage on the client side. HTML provides the structure and semantics of the content, while CSS handles the presentation and layout, ensuring a visually cohesive and appealing design. Together, these three technologies form the core building blocks of modern web development, allowing developers to create rich and interactive user experiences.


3 - Explain the difference between var, let, and const

- var: Function-scoped, hoisted, and can be redeclared and reassigned.
- let: Block-scoped, hoisted, and can be reassigned but not redeclared in the same scope.
- const: Block-scoped, hoisted, must be initialized during declaration, and cannot be reassigned
after initialization.
